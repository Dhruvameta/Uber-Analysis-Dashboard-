# Minimal Uber Dashboard

A clean and modern Uber dashboard built in **Microsoft Power BI**, designed with a focus on simplicity, clarity, and visual hierarchy. This project showcases an interactive analytics dashboard that presents key ride-hailing insights through a minimal and visually appealing design.

## Overview

This Power BI dashboard provides a comprehensive overview of Uber's operational and business metrics. It combines clean visualizations with an intuitive layout, making it easy to monitor performance and identify trends at a glance.

## Features

- Built using Microsoft Power BI
- Minimal and modern dashboard design
- Booking analytics
- Revenue overview
- Monthly booking and revenue trends
- Payment method analysis
- Distance travelled statistics
- Average trip distance
- Lost bookings summary
- Top pickup and drop locations
- Driver and customer ratings

## Dashboard Insights

The dashboard includes:

- Total Bookings
- Total Revenue
- Distance Travelled
- Average Distance
- Lost Bookings
- Monthly Booking Trends
- Monthly Revenue Analysis
- Revenue by Payment Method
- Top Pickup & Drop Locations
- Driver & Customer Ratings

## Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX
- Data Visualization

## Design Goals

This project was created to:

- Practice Power BI dashboard development
- Explore modern dashboard design principles
- Improve information hierarchy and visual storytelling
- Build a clean, portfolio-ready analytics dashboard
